#About the Project

Kowser Ahmed XP is a browser-based interactive experience that recreates the classic Windows XP desktop environment. It was built as a creative portfolio piece and technical showcase, demonstrating how nostalgic UI systems can be reimagined on the web.

#Asset Disclaimer

This project is designed as a parody and for educational demonstration only. It reinterprets a well-known operating system interface for learning and creative exploration.

All visual elements such as icons, sounds, and branding associated with Windows XP belong to Microsoft Corporation. No ownership is claimed over these assets. Their inclusion is purely for commentary, nostalgia, and illustrative purposes.
